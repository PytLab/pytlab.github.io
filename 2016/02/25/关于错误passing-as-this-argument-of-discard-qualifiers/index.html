关于错误passing '...' as 'this' argument of '...' discard qualifiers

今天又碰到了个新问题，先上代码：
vect.cpp

vect.h

在与另外一个驱动程序一起编译的时候编译器总是报错：

弄得我真是头大，这error是哪里的问题？
可怜的我还是跑去爆栈上去找答案了，结果一搜就有，
传送： templates - C++ error: passing const as ‘this’ argument - Stack Overflow

原来是因为被声明成`const`的成员函数，只能在其中调用其他的`const`成员函数，不能是没有`const`关键字的成员函数！

我把函数都加上了const关键字，终于，C++的编译器终于不傲娇了！
